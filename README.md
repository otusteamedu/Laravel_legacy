### Уникальность функционала

DRY нарушается в Region и Locales, повторяющийся код по виду: 
```php
  switch ($locale) {
            case "ko_KR":
                $this->locale = new ko_KR();
                break;
            default:
                $this->locale = new ko_KR();
                break;
        }
```
