1. server.php
Файл сервера, запускается первым. 
Должны быть права на создание файла в этой же папке, будет создан файл для сокета server.sock
Основные настройки файла server.php:
    - $socketFile - название файла для сокета, по умолчанию server.sock
    - $transport - указываем транспорт, по умолчанию unix, если указываем tcp или udp, то в
        $socketFile обходимо указать ip адрес и порт, пример, 127.0.0.1:8000
    - $msg - сообщение
    - $sendMsg - сообщение, которое будет отправлено клиенту, состоит из номера и $msg
    - $i - с какого числа начинается порядковый номера
    - $finish - кол-во оправленных сообщений
    - $timeWait - с каким периодом будут отправлены сообщения, измеряется в секундах

2. client.php
Файл клиента, запускаем вторым после server.php!!!
Настройки:
    $socketFile - название файла для сокета, по умолчанию server.sock. Необходимо, чтобы совпадал 
    с файлом для сокетов в server.php
    $revMsg - тветное сообщение для сервера, состоит из $revMsg + сообщение, которе пришло от сервера.
